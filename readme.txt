The Resolution Toolkit: a plugin by Kopatheme

== Description ==

The Resolution Toolkit plugin is an easy way to get latest tweet from the tweet widget.

== Installation ==

1. Upload the files to the /wp-content/plugins/resolution-toolkit/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Appearance->Widgets, pull widget (Resolution) Tweet to sidebars, and use.
